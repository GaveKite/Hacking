[House Cleaning] : 자신이 해킹에 사용했던 흔적들을 지우는 것 (매우매우매우매우매우 중요함)

[로그클리닝] : 서버가 시스템에 관한 행위를 기록하는 파일 = 로그
		로그 파일의 위치 = cd /var/log
		secure : 원격 셸에 관한 사용기록
		dmesg : 부팅 시 기록되는 로그
		message : 시스템 콘솔의 결과 /로그인 설정/ 장치정보 로그인 기록
		
		+ ls /var/run/utmp : 현재 시스템에 로그인한 사용자의 상태(바이너리)
		계정 정보 / 시간 / 어떤상태인지

		lastlog : 사용자들이 로그인한 정보를 담고 있음
		httpd 디렉토리 : 웹서버에 요청된 기록들을 담고 있음 => 공격자의 IP, 날짜가 담겨있음 -> 해커가 지워야할 중요한 정보들 중 하나.
